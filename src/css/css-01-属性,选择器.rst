选择器
========

通用选择器
-----------

:code:`*` 匹配所有的

.. code-block:: css

  * {}         /* 匹配文档中所有元素 */
  div > * {}   /* 匹配div 下所有的元素 */

选择器优先级的计算方法
-----------------------

1. 计算ID选择器的数目(=a)
2. 计算类选择器, 属性选择器, 伪类选择器 的数目(=b)
3. 计算 元素选择器 和伪元素选择器 的数目(=c)
4. 忽略通用选择器.
